# mytest

Progetto Python di esempio creato rapidamente.

Requisiti:
- Python 3.8+
- `pip` incluso

Istruzioni rapide (Windows PowerShell):

1. Crea e attiva un virtual environment:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

2. Installa dipendenze:

```powershell
pip install -r requirements.txt
```

3. Esegui l'app di esempio:

```powershell
python -m src.myproject.main
```

4. Esegui i test:

```powershell
pip install pytest
pytest
```

Vuoi che aggiunga anche un `pyproject.toml` o configurazione `setup.cfg`/`tox` ora?
